Problem Definition The MulpX platform is designed to enable user to initiate multiple transactions from one dashboard. MulpX would solve the problem of recurrent transfers, through the scheduled payments interface. Users can now execute financial transactions to multiple recipients at a click of a button. With this in mind, the solution was designed to simplify the financial transaction process as well as an easy to use interface, also we intend using the application to reduce the circulation of cash by introducing a transaction wallet feature where people with mobile device can send money without their account number but with the transaction id on the application.
